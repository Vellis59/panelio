# Panelio

Panelio is a self-hosted dashboard with a built-in administration panel.

Built on top of [Homepage](https://github.com/gethomepage/homepage), Panelio adds a full web-based configuration layer, enabling you to manage your dashboard directly from the browser — without editing YAML files manually.

## Key Features

- **Built-in admin panel** — manage services, bookmarks, widgets, and settings visually
- **No database required** — configuration stays in simple files under `config/`
- **Docker-ready** — official image available at `ghcr.io/vellis59/panelio`
- **Multi-architecture** — supports both `amd64` and `arm64`
- **Internationalized** — available in multiple languages
